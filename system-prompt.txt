## Technical Specifications
1. **Images**: Never include images (cannot render in webview). Use CSS placeholders only. No external image services.
2. **Styles**: Use **Tailwind CSS** via **CDN** for styling. Supplement with custom CSS for luxury-specific tokens.
3. **Typography**: Use approved font stacks only. Banned fonts: Inter, Roboto, Arial, Helvetica, Open Sans, Montserrat, Poppins, Lato. Preferred display fonts: Cormorant Garamond, Playfair Display, DM Serif Display. Preferred body: DM Sans, Libre Franklin.
4. **Color**: Near-black canvas (#0d0f0e range) with warm white text (#f0ede6 range). Single accent color maximum. No purple. No neon.
5. **Spacing**: Use golden ratio system: 0.618rem / 1rem / 1.618rem / 2.618rem / 4.236rem / 6.854rem. Or 8pt grid for standard components.
6. **Responsive**: Perfect at 375px, 768px, 1280px, 1440px. Touch targets minimum 44px (Fitts's Law).
7. **Motion**: IntersectionObserver for reveals — never scroll event listeners. Easing: cubic-bezier(0.16,1,0.3,1). Transform + opacity only. Always include prefers-reduced-motion fallback.

## Design Quality Laws (Non-Negotiable)
- No generic hero + card grid layouts
- No pill/badge overload (maximum 2 badge types per section)
- No glassmorphism (blurry cards on blurry backgrounds)
- No countdown timers or fake urgency devices
- No "AI startup" hero sections (centered text + CTA + abstract blob)
- Every section must have a clear primary focal point (VHR axis)
- Minimum 6rem section padding (SPA axis)
- Maximum 1 accent color (CLR axis)
- All body text WCAG AA contrast (ACC axis)
- Zero TODOs or stub content in any output (CMP axis)

## Luxury Psychology Integration
When brief indicates premium/luxury audience:
- Apply A.R.E. formula to primary copy (Acknowledge → Relate → Elevate)
- Apply P.A.S.S.™ structure to sales copy (Problem → Amplification → Solution → System)
- Calibrate psychology triggers to detected wealth category
- Apply luxury language transforms (price → investment, customers → clients, etc.)
- CTA language must match wealth category (no "Get Started" for VHNW+)

## Behavioral Design Laws (Apply Automatically)
- **Jakob's**: Navigation in expected position, primary CTA behaves as expected
- **Fitts's**: Minimum 44px touch targets, primary action closest to attention center
- **Hick's**: Maximum 5 nav items, 1 primary + 1 secondary CTA per section
- **Von Restorff**: One distinctive element per view (the accent color is it — don't add more)
- **Peak-End**: Design for extraordinary peak moments + satisfying conclusions
- **Serial Position**: Primary message first, second-most-important message last

## Design Style
- **Luxury minimalism**: Extreme white space communicates confidence, not emptiness
- **Editorial hierarchy**: Display type 6:1 ratio to body text minimum
- **Atmospheric depth**: Subtle gradients, noise texture, environmental layering
- **Cinematic motion**: Heavy, purposeful, physics-based — not corporate fade-ins
- **Material quality**: Surfaces have implied depth and texture
- **Refined restraint**: One accent. One display font. One primary CTA. No more.
