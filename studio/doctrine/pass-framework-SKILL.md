# SKILL: P.A.S.S.™ COPYWRITING FRAMEWORK

**Use this skill whenever:** BASS is writing any copy, RALPHY needs copy scaffolding for
a design, or any agent is producing written content for Kupuri/Querencia deliverables.
P.A.S.S.™ is the canonical copywriting structure for luxury markets.

Anti-pattern: Do not use generic copywriting frameworks (PAS, AIDA, FAB) for luxury
audiences. These frameworks optimize for volume conversion, not premium conversion.
P.A.S.S.™ is engineered for sophistication — the difference is the System layer.

---

## THE FOUR LAYERS

### P — PROBLEM
**What it does:** Names the friction with precision and sophistication.

**Luxury copywriting rule:** The problem statement must make the reader feel understood,
not afraid. Generic copywriting amplifies fear. Luxury copywriting demonstrates insight.

**Structure:**
```
[Specific friction] + [specific context where it manifests] + [the real cost]
```

**Examples by wealth category:**

AFFLUENT:
```
Generic: "Struggling to get design work done on time?"
P.A.S.S.™: "Creative projects that begin well tend to collapse in revision rounds —  
            not because the work is bad, but because the brief was never precise enough 
            to evaluate against."
```

HNW:
```
Generic: "Tired of agencies that don't understand your brand?"
P.A.S.S.™: "The real cost of the wrong creative partner isn't the project budget.
            It's the two quarters of stakeholder time absorbed by revisions that  
            never resolve because there's no objective scoring framework."
```

VHNW:
```
Generic: "Looking for something truly unique?"
P.A.S.S.™: "When you've worked with every major studio, the problem isn't quality.  
            It's that the process itself has become predictable — and predictable 
            process produces predictable output, regardless of the talent involved."
```

---

### A — AMPLIFICATION
**What it does:** Shows the downstream cost of inaction — elegantly, not alarmingly.

**Luxury copywriting rule:** Do not use fear-based amplification. Affluent audiences
respond to it; HNW+ audiences see it as manipulation. Amplify by articulating the
opportunity cost, not the disaster scenario.

**Structure:**
```
[What continues to happen] + [what that forecloses] + [what remains at stake]
```

**Examples:**

AFFLUENT (fear-adjacent amplification is acceptable):
```
"Every month without a conversion-optimized presence is market share that compounds 
toward competitors who have already solved this. The gap widens, not narrows."
```

HNW (opportunity cost amplification):
```
"Each iteration cycle that doesn't resolve adds senior hours to a problem  
that could be closed in a single session with the right brief architecture.  
The compounding cost is invisible until it appears on a salary report."
```

VHNW (consequence amplification):
```
"Generic creative output in your market signals that the brand hasn't been 
curated at the level your audience expects. At this segment, that signal  
is permanent — first impressions in luxury are not revised."
```

UHNW (legacy amplification only):
```
"The communication infrastructure you build now will be the first chapter  
of the institutional record your family stewards for the next generation."
```

**Amplification DO NOT list:**
```
✗ "Don't let this happen to you..."
✗ "Warning: [scary statistic]"
✗ "Most businesses fail because..."
✗ "Before it's too late..."
✗ Countdown timers, artificial urgency devices
```

---

### S — SOLUTION
**What it does:** Presents the resolution with confidence and specificity.

**Luxury copywriting rule:** The solution statement must be specific to their problem,
not generic to the category. Never say "we offer premium design services." Say what
the specific mechanism is and why it resolves the specific friction named in P.

**Structure:**
```
[Named mechanism] + [how it specifically resolves the named problem] + [what that produces]
```

**Examples:**

Generic (DO NOT USE):
```
"We offer premium design services tailored to your needs."
```

P.A.S.S.™ solution (HNW):
```
"The Karpathy Council protocol produces three scored design directions simultaneously.  
Each scores against 14 objective axes — before a single stakeholder review hour is  
committed. You evaluate finished candidates, not rough concepts."
```

P.A.S.S.™ solution (VHNW):
```
"We replace the iteration spiral with a brief architecture layer that encodes your  
identity before any creative execution begins. The result is first-pass output that  
requires curation, not course correction."
```

**Solution architecture checklist:**
- [ ] Names the specific mechanism (not "our process" — the actual named thing)
- [ ] Connects mechanism directly to the named problem
- [ ] Describes the output state, not the activity
- [ ] No adjectives without support ("premium", "exceptional", "world-class" — banned unless proven)

---

### S — SYSTEM
**What it does:** Demonstrates the premium process — shows the experience is engineered,
not improvised. This is the differentiating layer that generic P.A.S. frameworks omit.

**Luxury insight:** High-net-worth buyers are buying a system, not a transaction.
The System layer communicates:
1. Your process is designed, tested, and non-arbitrary
2. The client experience is protected by structure, not dependent on luck
3. The relationship continues beyond the transaction

**Structure:**
```
[Step-by-step process with named stages] + [quality gates] + [ongoing relationship mechanism]
```

**Example (HNW context):**
```
"The engagement structure is three phases:

Discovery — A 60-minute brief architecture session that encodes your objectives,
identity constraints, and success metrics into a machine-readable brief.

Production — Three simultaneous creative directions built against the brief.
Each scored against 14 objective axes before reaching you.

Delivery — A scored output package with implementation specifications,
asset files, and a 30-day follow-on advisory window.

There are no revision rounds in this model. There is brief architecture and
scored output. One replaces the other."
```

**System layer for each wealth category:**

AFFLUENT — Show the process clearly with milestones and deliverables
HNW — Show the process but emphasize what it protects (their time, their principals)
VHNW — Show the methodology behind the process, not just the steps
UHNW — Describe the relationship architecture, not the process steps

## COMPLETE P.A.S.S.™ EXAMPLE

**Context:** HNW founder, $15M-stage company, needs complete brand/web redesign

```
PROBLEM:
At growth stage, the creative infrastructure built in years 1-3 becomes  
a constraint. Visual identity that worked for a $2M company signals the  
wrong things at $15M — not because it's bad design, but because the audience  
has changed and the design hasn't.

AMPLIFICATION:
The cost isn't the outdated website. It's the enterprise conversations that  
stall because the credibility signal doesn't match the contract size being  
discussed. A single deal lost to a perception gap costs more than a complete  
rebrand. Most companies discover this too late, after the opportunity has closed.

SOLUTION:
The Identity Architecture engagement begins with a 3-hour brand deconstruction  
session that maps your current perception gap against your growth targets.  
The output is a scored identity system — not a logo file, but a machine-verified  
coherence framework that every creative decision is measured against from day one.

SYSTEM:
Phase 1 (Week 1): Brand Audit + Perception Gap Analysis
— UDEC-scored assessment of current identity against growth-stage benchmarks  
— Competitor signal mapping for your specific market position  
— Brief architecture encoding all identity constraints

Phase 2 (Weeks 2-4): Identity System Development  
— Three parallel directions developed simultaneously  
— Each scored before presentation (UDEC composite floor: 8.5)  
— You review finished systems, not mood boards

Phase 3 (Week 5): Implementation + Handoff  
— Full asset library + implementation specifications  
— 90-day advisory relationship for deployment decisions  
— Memory encoding for future creative consistency

Investment: From $28,000. Scope determined by brand complexity.
Timeline: 5 weeks from signed agreement.
```

## COPY QUALITY SCORING

When LENA scores CON (Content Quality), apply this P.A.S.S.™ checklist:

### Score 9.0+ (Exceptional P.A.S.S.™ implementation)
- All four layers present and distinct
- Problem is specific and insightful (not generic)
- Amplification is opportunity-cost framing (not fear-based) for HNW+
- Solution names the specific mechanism
- System shows engineered process with named phases
- Language register perfectly matched to wealth category
- Zero luxury language pollution

### Score 8.5–9.0 (Solid P.A.S.S.™)
- All four layers present
- Problem and amplification are specific
- Minor register mismatches tolerated

### Score < 8.5 (P.A.S.S.™ incomplete)
- Missing any layer → automatic flag
- Generic problem statement → flag
- Fear-based amplification for HNW+ → flag
- Solution without named mechanism → flag
- System described as steps without named phases → flag

## BANNED COPY PATTERNS

Permanently banned from all Kupuri/Querencia copy:

```
✗ "World-class" — empty superlative, earned by nobody
✗ "Cutting-edge" — SaaS marketing language
✗ "Passionate about [anything]" — signals inexperience
✗ "We're different because..." — makes you sound like everyone else
✗ "End-to-end solution" — corporate jargon
✗ "Synergy" — always banned, in every context
✗ "Leverage" (as verb) — overused to meaninglessness
✗ "Game-changing" — press release language
✗ "Take your business to the next level" — never
✗ "Don't miss out" — retail urgency tactic
✗ "Act now" — banned for all luxury segments
✗ Any sentence beginning with "We believe..." unless immediately followed by proof
```
